# Predict Test Scores [SIMPLE]

This is a super simple linear regression project I made to get comfortable with the basics of machine learning.

## What It Does

The model tries to predict a student's test score based on how many hours they studied.

- I used `scikit-learn` to create a linear regression model.
- I split the data into a training set and a test set.
- I trained the model.
- Then I plotted the results to see if the line made sense.

## Concepts
- `.fit()` and `.predict()`
- `train_test_split`
- `model.score()` for R²
- How to plot your model with matplotlib

## Tools I Used

- Python
- NumPy
- Matplotlib
- Scikit-learn

## What It Looks Like

It plots a red regression line through the training data, and prints out how well the model performs (R² score).

## Why I Made This

I'm learning machine learning and wanted to start with something very basic, just to build confidence and get hands-on. This is one of my first notebook projects.
